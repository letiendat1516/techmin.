# Custom Business Software Landing Page

Single-page React + TailwindCSS landing page for a software development company that builds custom business management systems.

## Stack

- React (functional components)
- TailwindCSS
- Vite

## Project structure

- `src/App.jsx` - page composition
- `src/components/` - reusable section components
- `src/data/landingContent.js` - central content for services, products, process, and benefits
- `src/index.css` - Tailwind entry styles

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
